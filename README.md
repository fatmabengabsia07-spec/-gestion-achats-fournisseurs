# Gestion Achats & Fournisseurs

Application web de gestion des achats et fournisseurs.

## Stack technique
- **Backend** : Spring Boot 3 + MySQL
- **Frontend** : Angular 17
- **DevOps** : Docker, GitHub Actions, Kubernetes, ArgoCD
- **Monitoring** : Prometheus + Grafana

## Lancer avec Docker
```bash
docker compose up --build
```

## Accès
- Frontend : http://localhost:4200
- Backend API : http://localhost:9090/api
- Swagger : http://localhost:9090/swagger-ui.html
